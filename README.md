# 🤖 Self-Correcting Multi-Agent Code Generator

A lightweight, zero-storage, multi-agent Python development ecosystem built using **CrewAI** and **OpenRouter**. This project features a two-agent crew (a Senior Programmer and a Senior QA Tester) working in an autonomous execution loop to write, run, debug, and verify Python code completely on their own.

---

## 🚀 How It Works

The system utilizes an agentic feedback loop to ensure generated code actually runs successfully before presenting it to the user.

1. **The Programmer Agent** receives a user prompt and utilizes chain-of-thought reasoning to design and write a clean Python script.
2. **The QA Tester Agent** intercepts the script and utilizes a **custom local execution tool** (`subprocess`) to safely run the script in a sandbox.
3. **The Self-Correction Loop:** If the code crashes or has a syntax error, the QA Tester captures the exact terminal stack trace and feeds it back to the Programmer. The Programmer rewrites the code, and the Tester runs it again. This continues until it returns a `Success!` status.

---

## ⚡ Features

* **Zero-Storage Footprint:** Runs cloud-hosted LLMs via OpenRouter, making it perfect for resource-constrained or low-disk-space environments.
* **Autonomous Debugging:** The agents fix their own code errors without human intervention.
* **Interactive CLI:** Prompt the crew with any programming task right from your terminal.
* **Safety Sandbox:** Includes a `timeout` safeguard on the local execution engine to violently break out of accidental infinite loops generated by the AI.

---

## 🛠️ Tech Stack & Model Selection

* **Framework:** CrewAI
* **LLM Engine:** `z-ai/glm-4.5-air:free` (via OpenRouter) — Chosen for its highly efficient Mixture-of-Experts (MoE) architecture and robust native support for function calling/tool use.
* **Environment Management:** `python-dotenv`
* **Local Runner:** Python `subprocess` module

## 📦 Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/AstroCode10/micro-developer.git
cd micro-developer
```

### 2. Create a Virtual Environment (Optional but Recommended)
```bash
python -m venv venv
# On Windows:
.\venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install crewai python-dotenv
```

### 4. Configure Environment Variables

Create a file named .env in the root directory and add your OpenRouter API key:

```bash
OPENROUTER_API_KEY=your_actual_openrouter_api_key_here
```

---

## 📋 Usage

Launch the script directly from your terminal:

```bash
OPENROUTER_API_KEY=your_actual_openrouter_api_key_here
```

You will be prompted to enter a programming problem. For example:
**Enter the programming problem you want to solve:** Write a Python script that calculates the Fibonacci sequence up to the 10th element.
